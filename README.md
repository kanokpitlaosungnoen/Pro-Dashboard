# 📦 ProDashboard

A product and inventory management website designed for retail stores and wholesale businesses.
ProDashboard helps simplify product management, stock tracking, and sales monitoring through an all-in-one dashboard system.

---

## 🎯 Project Overview

ProDashboard is a web-based management system developed to organize product information and inventory within stores and warehouses.
The platform supports both retail and wholesale businesses, focusing on usability, fast data access, and reducing stock management errors such as out-of-stock issues or incorrect inventory counts.

---

## ✨ Features

* 📊 Business overview dashboard
* 📦 Product management system
* 🧾 Add / Edit / Delete products
* 📉 Inventory tracking system
* 🚨 Low stock alerts
* 📈 Sales analytics and statistics
* 👥 Customer and wholesale management
* 🔍 Product search and filtering
* 🌙 Dark Mode UI support
* 📱 Responsive design for multiple devices

## 🎨 Color Palette

The ProDashboard color palette was designed to create a modern, clean, and professional dashboard interface.
The design mainly uses blue, gray, and status colors to improve readability and help users understand information more efficiently.
